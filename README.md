## 🚀 Usage

### To Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/rounak161106/rounak161106.github.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd rounak161106.github.io
   ```

3. Open `index.html` in your web browser:
   ```bash
   open index.html
   ```
   Or simply double-click the file.
